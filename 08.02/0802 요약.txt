File -> FileReader -> BufferedReader(버퍼 = 성능향상)
File -> FileWriter -> BufferedWriter (버퍼기능, 텍스트를 한 라인씩)

File -> FileInputStream -> BufferedInputStream
File -> FileOutputStream -> BufferedOutputStream

File -> FileInputStream -> InputStreamReader -> BufferedReader
File -> FileOutputStream -> OutputStreamWriter -> BufferedWriter


File -> FileOutputStream -> PrintWriter (.write / .print)

File -> FileWriter -> PrintWriter

File -> FileInputStream -> DataInputStream (.writeBoolean, Byte, Int, UTF 등)

File -> FileInputStream -> ObjectInputStream (.readObject)


FileInputStream = 1byte씩 읽기, 한글 출력 불가
InputStreamReader = 1문자씩 읽기, 한글 출력 가능, 바이트스트림을 문자 스트림으로 변환
BufferedReader = 1라인씩 읽기, reader 타입만 매개변수로 가능
DataInputStream  = 기본 타입 데이터 입출력
PrintStream, PrintWriter = 줄바꿈 처리 및 형식화된 문자열 출력
ObjectInputStream, ObjectOutputStream = 객체 입출력
